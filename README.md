<h2>Description</h2>
<p>
  This project is able to run a live stream where the user will deal cards from a standard playing deck. It will be able to detect the suit and face value of the card. 
  It will then keep a running count based on the standard for counting cards in BlackJack to give the user advice on what move to make next. The user can input how many
  decks they are playing with and the model will be able to remove all cards seen already. 
</p>

<h2> Technical Stack </h2>
<p>
  This project was built using a Raspberry Pi 5 and Camera module.
  Programmed in Python. Website created using standard HTML and CSS. 
  OpenCV was used for the camera. 
</p>
